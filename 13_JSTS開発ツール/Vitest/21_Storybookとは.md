# Storybookとは

作成日 2025/11/11

## 1. 公式サイト（英語）を読む

[Storybook: Frontend workshop for UI development](https://storybook.js.org/)

> Storybook is a frontend workshop for building UI components and pages in isolation. Thousands of teams use it for UI development, testing, and documentation. It's open source and free.

## 2. Storybookをインストールする

[Install Storybook | Storybook docs](https://storybook.js.org/docs/get-started/install)

> Use the Storybook CLI to install it in a single command. Run this inside your project’s root directory

```bash
# 既存のプロジェクトに対して追加インストールする
cd {your-project}
npm create storybook@latest
```
