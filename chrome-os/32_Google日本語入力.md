# Google 日本語入力を使う

作成日 2020/03/18

## ユニコード表記を変換する

Google 日本語入力は Unicode 表記を変換できる。U+の後に 16 進数を続ける

| Before | After | Comment                                |
| ------ | ----- | -------------------------------------- |
| U+8332 | 茲    | 音読み=>シ・ジ、訓読み=>ここに・しげる |
| U+00A5 | ¥     | 円マーク                               |
| U+9A65 | 驥    | き、1 日に千里を走るほどの駿馬         |
