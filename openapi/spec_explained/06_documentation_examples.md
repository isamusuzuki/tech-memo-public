## The Documentation Fields

ほとんどのオブジェクトで、description キーが使える。ここに説明文を書く

description では、マークダウンが使える

- Literal mode ... `|` を使う。アウトプットも改行を維持する
- Folded mode ...  `>` を使う。アウトプットから改行を取り除く

## The Example Object

- example ... 一例の場合
- examples ... 複数例の場合
