# tsconfig.jsonルート項目

作成日 2025/09/09

## 解説記事を読む

[なんとなく使うtsconfig.jsonをやめる](https://zenn.dev/uniformnext/articles/e2106ba4d995b1)

ルート項目

- `include` ... コンパイル対象とするファイルやフォルダを指定
- `exclude` ... コンパイルから除外するファイルやフォルダを指定

compilerOptions項目

- `target` ... `ESNext`,`ES2020`,`ES5`
- `module` ... `ESNext`,`CommonsJS`,`ES6`
- `outDir`
- `rootDir`
- `strict`
