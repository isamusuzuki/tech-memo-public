# WSL 上にて、つつがなく GUI プログラムを使うためのルーティン

作成日 2021/03/22

## 事前準備

```bash
# geditをインストールする
sudo apt install gedit -y
```

## ルーティン

- Windows が起動したときに、X サーバーが自動でスタートするようには設定しない
- 使う直前に、デスクトップにある XLaunch アイコンをクリックする

```bash
echo $DISPLAY

echo $LIBGL_ALWAYS_INDIRECT

# 接続テスト
dbus-launch --exit-with-x11

# geditを起動してみる
gedit &
```
