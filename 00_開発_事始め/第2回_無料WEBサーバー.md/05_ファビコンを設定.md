# ファビコンを設定する

作成日 2026/01/08

## 1. 前提条件

縦幅・横幅が同じの正方形の、最大256x256くらいサイズの、PNG画像が用意できているものとする

## 2. Faviconジェネレーターを利用する

[Favicon ジェネレーター](https://favicon-generator.mintsu-dev.com/)に行く

画像をFaviconに変換するコーナー ＞ 「変換する画像を選択する」ボタンをクリック ＞ PNG画像を選択

元画像エリアに指定した画像が表示される

16x16, 32x32, 48x48の3つにチェックを入れてから、「画像を変換」ボタンをクリック

変換後画像エリアに小さい画像が表示される ＞ 「ダウンロード」ボタンをクリック

favicon.icoファイルがひとつだけダウンロードされる

このファイルは「マルチアイコン」といって、ひとつのファイルに、3種類のサイズの画像が含まれている

## 3. HTMLファイルにファビコンの設定を追加する

publicフォルダの直下にfavicon.icoファイルをコピペする

head要素に、次の1行を追加する

```html
<!DOCTYPE html>
<html lang="ja">
    <head>
        <link rel="shortcut icon" href="/favicon.ico">
    </head>
    <body>
    </body>
</html>
```
