# 覚えておきたい、最低限のLinuxコマンド

作成日 2026/01/08

```bash
# ディストリビューションを調べる
cat /etc/os-release

# 今、自分がどのディレクトリにいるかを知る
pwd

# 今、自分がいるディレクトリの中身を表示させる
ls

# 今、自分がいるディレクトリの中身を、隠しファイルも含めて、表示させる
ls -a

# 今、自分がいるディレクトリの中身を、詳しく、表示させる
ls -l

# ひとつ下の階層のディレクトリ（サブディレクトリ）に移動する
cd {directory-name}

# ひとつ上の階層のディレクトリ（親ディレクトリ）に移動する
cd ..

# テキストファイルの中身を表示させる
cat {file-name}
```

## おまけ：ちょっと高度な使い方

sudoするたびにパスワードの入力を求められるのが面倒だなと感じたとき

```bash
sudo visudo
# => 最終行に以下を追加する
# => YOURNAME ALL=(ALL) NOPASSWD: ALL
```
