# C3とは

作成日 2026/01/26

C3 = create-cloudflare CLI

公式ドキュメント => [https://developers.cloudflare.com/pages/get-started/c3/](https://developers.cloudflare.com/pages/get-started/c3/)

Cloudflare向けプロジェクトを作成するツール

```bash
npm create cloudflare@latest
```

- node,npmがインストールされている以外の事前準備は不要
- create-cloudflareパッケージをインストールするところから始まる
- 質問に答える形で、希望のプロジェクトが作成される
