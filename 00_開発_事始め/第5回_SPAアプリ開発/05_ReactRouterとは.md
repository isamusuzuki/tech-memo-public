# React Routerとは

作成日 2026/01/27

公式サイト => [React Router Official Documentation](https://reactrouter.com/)

Reactで構築されたシングルページアプリケーション（SPA）において、URLとコンポーネントをマッピングしてページ遷移（ルーティング）を管理するライブラリ

- SPAでのルーティング実装: URL（/home, /about）に応じて、表示するコンポーネントを瞬時に切り替える。
- ページ全体のリロードなし: `<a>`タグの代わりに`<Link>`や`<NavLink>`を使用することで、リロードなしに遷移。
- 履歴の管理: ブラウザの「戻る・進む」ボタンや履歴管理（ブラウザヒストリーAPI）とシームレスに連携。
- ネストされたルーティング: 画面内の特定のエリアだけを入れ替えるような、階層的なコンポーネント構成が可能。
- 動的ルーティング: URLパラメータ（/user/:id）を使用して、ユーザーIDに基づいたコンテンツを表示。

| 技術            | 説明                       | 範囲                       |
| --------------- | -------------------------- | -------------------------- |
| React           | JSフレームワーク           | クライアントサイドのみ     |
| React Router    | ルーティングライブラリ     | クライアントサイドのみ     |
| Next.js         | フルスタックフレームワーク | React + ルーティング + SSR |
| Remix           | フルスタックフレームワーク | React + ルーティング + SSR |
| React Router v7 | Remixと統合された          | ルーティング + SSR         |

SSR = Server-Side Rendering
