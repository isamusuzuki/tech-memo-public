# システムアリアスとは

作成日 2021/03/26

よく使うコマンドへのショートカットを、システムアリアスという

```bash
!python --version
# Python 3.7.10

!cat /etc/os-release
# NAME="Ubuntu"
# VERSION="18.04.5 LTS (Bionic Beaver)"

!pip list
# Package                       Version
# ----------------------------- --------------
# ...
# pandas                        1.1.5
# ...
```
