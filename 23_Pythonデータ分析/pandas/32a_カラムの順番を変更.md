# カラムの順番を変更する

作成日 2025/11/25

## 1. 解説記事を読む

[DataFrameのカラムの順番を変える方法 | Kamiki](https://kamiki.blog/python-dataframe-reindex-insert/)

```python
# 元データのDataFrameを作成する。
df = pd.DataFrame(
    data=[[0, 1, 2], [3, 4, 5], [6, 7, 8]],
    index=['X', 'Y', 'Z'],
    columns=['A', 'B', 'C'],
)

print(df)
#    A  B  C
# X  0  1  2
# Y  3  4  5
# Z  6  7  8

# columns引数でカラムの順番を変える。
df_new = df.reindex(columns=['B', 'C', 'A'])

print(df_new)
#    B  C  A
# X  1  2  0
# Y  4  5  3
# Z  7  8  6
```
