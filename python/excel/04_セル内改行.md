# セル内改行を実現する

作成日 2022/05/24

```python
import openpyxl as xl
from openpyxl.styles import Alignment

wb = xl.Workbook()
ws = wb.worksheets[0]
ws.title = 'Sheet1'

ws['A1'].alignment = Alignment(wrap_text=True)
ws['A1'].value = 'abc\ndef\nghi\njkl'

excel_file = 'temp/test.xlsx'
wb.save(excel_file)
```
