# 辞書を操作する

作成日 2023/03/07

## 要素を追加する、要素を上書きする

```python
d = {'a': 1, 'b': 2}

d['c'] = 3

print(d)
# => {'a': 1, 'b': 2, 'c': 3}

d['a'] = 10

print(d)
# => {'a': 10, 'b': 2, 'c': 3}
```

## 要素を削除する

```python
d = {'a': 1, 'b': 2}

removed_value = d.pop('a')

print(removed_value)
# => 1

print(d)
# => {'b': 2}

del d['b']

print(d)
# => {}
```

## 辞書を連結する

```python
d1 = {'a': 1, 'b': 2}

d2 = {'a': 10, 'c': 3, 'd': 4}

d1.update(d2)

print(d1)
# => {'a': 10, 'b': 2, 'c': 3, 'd': 4}
```
